About the Model

System: Neuro-Symbolic (NeSy) pipeline for drug category prediction based on gene expression in brain regions.

  • Architecture: Modular, 7 main tasks (gene expression, drug targets, ATC hierarchy, feature integration, CNS classifier, ATC classifier, evaluation).
  • Data sources: DrugBank XML, DRUGseqr.gmt, single-cell RNA-seq brain data.
  • Stage 1: CNS/Non-CNS classifier (Gradient Boosting, accuracy: 0.89).
  • Stage 2: ATC category classifier (Random Forest, accuracy: 0.47, 14 classes).
  • Explainability: Top-k predictions, feature importance, brain evidence metrics (BES, BSR, CNS score), gene coverage, closest drugs, and reasoning steps.
  • Outputs: All results and explanations are consistent between the web interface and CLI.

For more details, see the PROJECT_REPORT.md in the repository.
